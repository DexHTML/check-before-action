# check-before-action

SEO-MVP проекта «Проверь перед действием».

## Суть

Статический сайт на GitHub Pages с быстрыми проверками рискованных ситуаций:

- подозрительный звонок;
- просьба о переводе денег;
- ссылка на доставку или оплату;
- код из СМС;
- защита родителей и родственников.

Проект не является официальной юридической, банковской или правоохранительной службой. Цель MVP — проверить SEO-спрос и поведение пользователей без вложений, серверов, PHP, базы данных и платёжных систем.

## Технологии MVP

- HTML
- CSS
- JavaScript
- GitHub Pages

## Структура

- `index.html` — главная страница
- `pages/` — SEO-инструкции
- `tools/` — интерактивные проверки
- `assets/css/` — стили
- `assets/js/` — логика инструментов
- `robots.txt` и `sitemap.xml` — базовая SEO-инфраструктура

## Следующие шаги

1. Включить GitHub Pages из ветки `main`, папка `/root`.
2. Проверить сайт по адресу `https://dexhtml.github.io/check-before-action/`.
3. После проверки добавить Яндекс Метрику.
4. Отправить sitemap в Яндекс Вебмастер и Google Search Console.
