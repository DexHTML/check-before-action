# Product layer

## Что усилено

После SEO-запуска усилены интерактивные инструменты, чтобы пользователь не просто прочитал страницу, а получил понятный следующий шаг.

## Изменения

### Проверка подозрительного звонка

Файл:

`assets/js/scam-call-checker.js`

Добавлено:

- блок «Следующий шаг» после результата;
- ссылки на срочные инструкции по ситуации;
- кнопка копирования ссылки на проверку;
- ссылка на карту сайта со всеми инструкциями;
- отдельные события Метрики для переходов из результата.

### Проверка просьбы о переводе

Файл:

`assets/js/money-request-checker.js`

Добавлено:

- блок «Что открыть дальше» после результата;
- подбор связанных инструкций по ответам пользователя;
- кнопка копирования ссылки на проверку;
- ссылка на карту сайта;
- отдельные события Метрики для переходов из результата.

### Генератор семейного кодового слова

Файл:

`tools/semeynoe-kodovoe-slovo.html`

Добавлено:

- блок «Что сделать дальше» после генерации памятки;
- кнопка копирования ссылки на инструмент;
- ссылка на инструкцию для родителей;
- усиленный сценарий: скопировать памятку → отправить близким → обсудить правило голосом или лично.

### Общий JS

Файл:

`assets/js/main.js`

Добавлено:

- универсальное копирование текущего URL по атрибуту `data-copy-current-url`;
- событие Метрики для копирования ссылки.

## Зачем это нужно

- Повышает шанс, что пользователь перейдёт из инструмента в релевантную инструкцию.
- Повышает полезность сайта после прохождения проверки.
- Помогает отслеживать не только посещения, но и намерение: копирование ссылки, переходы из результата, использование памятки.
- Поддерживает будущую монетизацию: сначала измеряем интерес к сценариям, потом решаем, какие комплекты делать.

## Что проверять в Метрике

- `call_checker_finished`
- `money_checker_finished`
- `family_code_generated`
- `call_result_copy_url`
- `money_result_copy_url`
- `family_tool_copy_url`
- переходы из результатов в связанные инструкции
- копирование семейной памятки

## Следующий продуктовый кандидат

После первых данных можно сделать отдельный бесплатный лид-магнит:

- «Памятка для родителей в 7 пунктах»;
- «Чек-лист безопасной сделки с рук»;
- «Чек-лист безопасного платежа для ИП».

Пока не включаем оплату и сбор данных.
