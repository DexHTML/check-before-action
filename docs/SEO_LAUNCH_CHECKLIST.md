# SEO launch checklist

## Готово

- GitHub Pages опубликован.
- Главная страница создана.
- 3 интерактивных инструмента созданы:
  - проверка подозрительного звонка;
  - проверка просьбы о переводе;
  - генератор семейного кодового слова.
- 6 SEO-страниц первой очереди созданы.
- `robots.txt` создан.
- `sitemap.xml` создан.
- `.nojekyll` добавлен.
- `404.html` добавлен.

## Следующий ручной этап

1. Подключить Яндекс Метрику.
2. Добавить ID счётчика в проект.
3. Добавить сайт в Яндекс Вебмастер.
4. Добавить sitemap:
   `https://dexhtml.github.io/check-before-action/sitemap.xml`
5. Добавить сайт в Google Search Console, если доступно.
6. Проверить индексацию через 7–14 дней.

## Метрики, которые смотрим через месяц

- сколько страниц в индексе;
- есть ли показы;
- есть ли клики;
- какие запросы появились;
- какие страницы получают показы;
- запускают ли пользователи инструменты;
- доходят ли до результата;
- нажимают ли кнопки копирования/интереса.

## Важное

Проект пока не коммерческий. Оплату, PHP, базу данных, серверную админку и платные комплекты добавляем только после первых признаков спроса.
