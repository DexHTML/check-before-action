Конфиденциальность

Политика конфиденциальности

Коротко: сайт не просит вводить номера карт, коды из СМС, пароли, паспортные данные или другие секретные данные. Интерактивные проверки работают в браузере.

Какие данные вводить нельзя

На сайте нельзя вводить коды из СМС, пароли, PIN-коды, данные банковских карт, паспортные данные, адреса, реальные номера телефонов и другую чувствительную информацию. Если где-то в будущем появится форма обратной связи, она должна использоваться только для обычного сообщения без секретных данных.

Как работают проверки

Текущие инструменты сайта работают на JavaScript прямо в браузере пользователя. Они используют выбранные ответы, чтобы показать уровень риска и рекомендации. Результаты проверок не отправляются в собственную базу данных сайта, потому что на текущем этапе базы данных нет.

Аналитика

На сайте подключена Яндекс Метрика для оценки посещаемости, кликов и работы страниц. Метрика может использовать cookies и технические данные браузера в соответствии со своими правилами. Эти данные нужны, чтобы понять, какие страницы и инструменты полезны пользователям.

GitHub Pages

Сайт размещён на GitHub Pages. Это означает, что техническая доставка страниц осуществляется инфраструктурой GitHub. На этапе MVP сайт является статическим и не имеет собственной серверной базы данных.

Внешние ссылки

На сайте могут быть ссылки на официальные сервисы, поисковые системы, банки, справочные материалы или будущие формы обратной связи. После перехода по внешней ссылке действуют правила соответствующего сервиса.

Изменения политики

Если проект будет развиваться, появятся формы, платные материалы, серверная часть или личные кабинеты, политика конфиденциальности должна быть обновлена до запуска таких функций.

Напоминание: если вы уже сообщили код, данные карты или перевели деньги, действуйте через официальный банк и соответствующие службы, а не через случайные формы в интернете.
